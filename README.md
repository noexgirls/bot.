# Meeting Management Telegram Bot

Telegram-бот для управления совещаниями: создание, редактирование, перенос и оповещение участников.

## Основной функционал

### Управление совещаниями
- Создание совещаний
- Изменение данных совещания
- Удаление совещаний
- Перенос совещаний на другую дату/время
- Добавление описания к совещанию

### Повестка
- Добавление повестки совещания
- Редактирование пунктов повестки

### Уведомления
- Оповещение участников о создании совещания
- Уведомления об изменениях и переносе

### Административная панель
- Отдельный админ-доступ
- Управление всеми совещаниями
- Контроль повесток и участников

---

## Технологии
- Python 3
- aiogram (Telegram Bot API)
- TinyDB
- python-dotenv
